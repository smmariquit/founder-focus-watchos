# Founder Focus watchOS App

A lightweight, distraction-free Apple Watch application built with SwiftUI to help founders stay focused.

## Features
- **Minimalist Pomodoro Timer:** 25-minute deep work cycles right on your wrist.
- **Glanceable Interface:** Large, monospaced digits designed for the Apple Watch screen.
- **Native SwiftUI:** Optimized for watchOS performance.

## Getting Started
1. Open this folder in Xcode.
2. Select the Watch App target.
3. Build and run on an Apple Watch Simulator or your paired device.
